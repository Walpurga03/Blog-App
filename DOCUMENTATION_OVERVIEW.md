## Überblick des Projekts

Die SoloMining Blog App ist eine moderne React-Anwendung mit folgenden Hauptfunktionen:

1. **Mehrsprachigkeit** - Unterstützung für Deutsch und Englisch mit i18next
2. **Themensystem** - Hell- und Dunkeltheme mit React Context API und CSS-Variablen
3. **Dynamische Blog-Inhalte** - Separierte UI-Übersetzungen und Blog-Inhalte für einfache Erweiterbarkeit
4. **Responsive Design** - Optimierte Darstellung auf allen Bildschirmgrößen

Das Projekt folgt modernen React-Best-Practices:

- Funktionale Komponenten mit Hooks
- Kontext-API für globalen Zustand
- TypeScript für Typsicherheit
- SCSS-Module für Styling
- Dynamisches Laden von Inhalten
